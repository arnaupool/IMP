This program was made with Python, because of that, you'll need Python for
your OS (https://www.python.org/downloads/).
Once downloaded, to execute the script open a terminal window and type in
'py imp.py'.

In order to select a file, you'll have to type in its name and file extension,
i.e.: cube.stl
